# Game Simple HTML

A premium 2D HTML5 game engine with modern UI/UX design.

## Features

- 🎮 Smooth player movement with physics
- ⌨️ Keyboard + Mouse controls
- 🎨 Cinematic main menu with animations
- ⚙️ Comprehensive settings panel
- 📱 Responsive design
- 🎯 Production-ready architecture

## Quick Start

```bash
# Open index.html in a modern browser
# Or use a local server:
npx serve .
```

## Controls

- **WASD / Arrow Keys**: Move player
- **Mouse**: Aim/Click interactions
- **ESC**: Pause menu
- **Enter**: Confirm selections

## Tech Stack

- HTML5 Canvas
- TypeScript (ES Modules)
- CSS Custom Properties
- Google Fonts (Plus Jakarta Sans, DM Sans)

## Project Structure

```
src/
├── core/       # Game engine core
├── entities/   # Game entities
├── ui/         # UI components
├── utils/      # Utilities
└── main.ts     # Entry point
```
